Detailed Description

Class that represents a bug.

This uses a standin class to provide a way to develop the adapter class first if you so choose.

You are not allowed to change this class in any way!

AFX_EXT_CLASS is a Microsoft directive that indicates that this class can be used outside the DLL. Do not change it or add it to any other classes.

std::shared_ptr<CBugStandin> CBug::mStandin
private

A stand-in bug class.

This draws simple stand-in graphics so you can see where the bug will be and what its state is. This makes it easy to develop code that uses the bug before you have the bug working.

And no, you may not change this to point to your bug implementation!
